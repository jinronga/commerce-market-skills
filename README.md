# commerce-market-skills

这个目录用于集中存放电商不同业务板块的 skills，覆盖跨境电商与国内电商场景。每个板块一个独立文件夹，文件夹内使用标准 `SKILL.md` 描述触发场景、执行流程和输出格式。

## Skills

| Skill | 板块 | 说明 |
| --- | --- | --- |
| `skills/ecom-scout` | 多平台选品 | 基于国家/地区热门关键词和平台搜索意图，挖掘适合 1688 供货、一件代发或轻库存测试的产品机会，并输出平台打法、成本售价、内容方向和风险。 |
| `product-hunting-expert` | 选品 | 用于跨境电商产品机会发现、蓝海/灰海判断、平台适配、利润空间和风险评估。 |
| `listing-optimization-expert` | Listing 优化 | 用于 Amazon、TikTok Shop/TK、Noon 等多平台产品标题、亚马逊五点描述、商品描述、核心关键词埋词和爆品标题优化。 |

## 建议后续板块

- `competitor-analysis-expert`：竞品分析
- `ads-growth-expert`：广告投放与增长
- `profit-calculator-expert`：利润测算
- `market-entry-expert`：国家/平台进入策略
